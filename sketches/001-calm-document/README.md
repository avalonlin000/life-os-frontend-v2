## Variant: 安静文档

### Design stance

把结衣前端当作一本持续更新的知识—现实转化文档：现实调用私人知识，知识形成方法，实践结果再改写个人系统。

### Key choices

- Layout: 手机优先单列长文，区块用分隔线而非卡片墙。
- Typography: 系统字体负责界面，衬线字体负责认知与整理长文。
- Color: 灰阶纸张低保真，不表达最终视觉。
- Interaction: 四页切换、知识方法确认、按需展开、实践状态变化和全局“记一笔”。

### Trade-offs

- Strong at: 看清知识来源、现实适配、方法形成和反馈更新的完整关系。
- Weak at: 同屏信息密度较低，桌面端没有充分利用横向空间。

### Best for

- 以手机阅读、长文字认知和安静回看为主的日常使用。
