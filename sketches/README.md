# 结衣四页低保真线框

> 2026-07-22 决策：正式只采用「安静文档」。聚焦工作台未采用，仅保留为设计过程参考。

> 同日方向修正：安静文档不再围绕“问题＋模型建议”组织，而是完整展示“现实调用私人知识 → 比较适配 → 形成并确认方法 → 实践检验 → 更新个人系统”。

这是一组一次性结构原型，不是正式前端代码，也不表达最终配色、字体或组件风格。

## 两种布局取向

| 维度 | 安静文档 | 聚焦工作台 |
|---|---|---|
| 核心取向 | 内容优先 | 操作与状态优先 |
| 手机阅读 | 强 | 中等 |
| 桌面信息密度 | 低 | 高 |
| 当前行动显著性 | 中等 | 强 |
| 长文字体验 | 强 | 中等偏强 |
| 快速记录 | 底部抽屉 | 居中弹层 |
| 最适合 | 日常手机阅读、认知长文 | 实践操作、桌面比较和管理 |

## 正式方向

手机和电脑统一使用“安静文档”的单列连续阅读结构。电脑端只增加留白与舒适阅读宽度，不吸收“聚焦工作台”的双栏、侧栏或控制台式布局。四页内容和全局“记一笔”保持一致。
