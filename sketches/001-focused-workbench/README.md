## Variant: 聚焦工作台

> 状态：未采用。仅保留为设计过程参考，不进入正式前端实现。

### Design stance

把结衣前端当作高密度操作工作台，主内容与当前状态、下一步和确认事项同屏出现。

### Key choices

- Layout: 桌面双栏＋侧栏，手机自动收为单列与底部导航。
- Typography: 全部使用系统字体，认知长文仅在正文中切换衬线体。
- Color: 灰阶低保真，不表达最终视觉。
- Interaction: 四页切换、状态按钮、折叠详情和居中“记一笔”。

### Trade-offs

- Strong at: 信息密度、桌面操作效率、同时理解当前状态与下一步。
- Weak at: 手机端视觉更像工具，安静阅读感弱于文档方案。

### Best for

- 需要频繁操作实践、比较信息，同时希望保留认知长文的使用方式。
