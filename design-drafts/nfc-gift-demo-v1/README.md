# NFC 礼物网页 Demo v1

## 定位

这是一个给 NFC 贴纸触发的单文件网页 Demo。当前 NFC 贴纸还没到，先把可部署到 GitHub Pages 或云服务器的网页体验做出来，后续只需要把页面 URL 写入 NTAG213 标签。

## 当前交付

- 文件：`index.html`
- 形态：单文件 HTML，无构建步骤
- 依赖：零外部依赖，适合 GitHub Pages / Nginx 静态托管
- 适配重点：iPhone Safari，使用 `100dvh`、点击后再请求麦克风/摄像头权限

## 七个 Demo

1. 通用平面 AR：桌面生长术
   - 不识别拼豆本身
   - 当前为 Safari 兼容点击放置降级版
   - 可打开后置摄像头，把爱心树放到桌面位置

2. 手势空间：隔空操控
   - 当前用触摸/鼠标模拟抓取、抛爱心、比心烟花
   - 后续可替换为 MediaPipe Hands

3. 人脸魔法镜：动态滤镜
   - 当前支持前置摄像头与星轨眼妆视觉层
   - 后续可替换为 MediaPipe Face Mesh 锚定

4. 运动光绘：魔法画笔
   - 当前用触摸/鼠标生成光绘轨迹
   - 停止后自动收束成爱心与文字
   - 后续可替换为摄像头帧差运动检测

5. 声纹花园：音波景观
   - 点击后请求麦克风授权
   - Web Audio 频谱驱动蓝塔、粉花、金粒子

6. 照片献花
   - 零权限保底
   - 上传照片后本地读取主色并生成同色花园

7. 触摸粒子画板
   - 零权限保底
   - 手指拖尾、点击烟花、停止后收成爱心

## NFC 落地步骤

1. 将本目录作为静态页面发布。
2. 获得公网 URL。
3. 用 NFC Tools App 写入 NTAG213 标签。
4. iPhone 解锁后碰 NFC 标签打开页面。
5. 若摄像头/麦克风权限被拒，仍可用照片献花和触摸粒子两层体验。
