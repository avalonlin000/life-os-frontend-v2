# JY-V4 三层职责与前端工作台产品重审

## 任务

- 将钧钧确认的专家审计结论写回结衣 PRD 与 SSD，停止从单次对话临时推导产品结构。

## 关键结论

- “认识世界＋改造世界”总纲保持不变，现实课题继续作为唯一组织中心。
- 新增三层职责：后台认知层负责带证据、可修正的长期观察；结衣 Agent 负责自然交流、主动提醒和现实验证建议；前端是长内容、高密度操作、快速记录、搜索比较、结果验证和纠错确认的工作台。
- 长期观察不能声称读取钧钧的“真正状态”；心理学与哲学只用于提出工作假设，单次信号不得生成心理或人格结论。
- 前端不展示人格评价，但不能降为只读档案；必须持续承载认知与学习材料、方法与实践、反馈更新以及高效率输入。
- 现有现实/知/行/思/道五入口是兼容实现，不再视为最终导航；下一轮先逐项做保留、合并、转 Agent、转后台、历史保留或删除审计。
