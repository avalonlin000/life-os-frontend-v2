# 结衣私人系统投资路线

> 当前只按“是否帮助钧钧认识世界、改造世界，并让实践结果继续更新世界观和方法论”排投资优先级，不按页面、接口或技术模块排路线。

## 已确认

- [x] M0：确认私人系统投资总纲与外部项目组合
- [x] M0.5：确认完整产品顶层设计，并按新总纲完成现有项目与结衣 Agent 的宏观审计

## 当前阶段

- [ ] M1：证明一个真实生活问题能够走通“认识现实 → 形成方法 → 实践改造 → 结果反馈 → 更新认识”的完整循环

## 后续阶段

- [ ] M2：以知识线发展“认识世界”，接入岁月时间管理线发展“改造世界”，证明两侧能够形成循环
- [ ] M3：让现实反馈持续修订世界观、方法论和各条系统线，并由结衣协调冲突与整体节奏
- [ ] M4：让多条系统线跨越整个人生形成长期的“认识—改造—再认识”复利

## 投资优先级

1. **认识现实**：先看清事实、矛盾、客观条件和可改变部分，避免用计划或情绪代替现实。
2. **形成方法**：吸收外部知识并结合实际，形成能够改变现实的方法论。
3. **进入实践**：方法必须真实作用于生活，不能停留在收藏、理解或任务列表。
4. **反馈更新**：用实践结果修订世界观、方法论和各条系统线。
5. **长期协同**：只有前四项产生真实价值后，才投资主动发现、高级自动化和视觉包装。

## 暂不投资

- 通用化、多用户、商业化能力。
- 第二套知识库、第二套 Agent 入口或平行产品主线。
- 复杂任务管理、连续打卡、积分和大屏仪表盘。
- 在 M1 真实价值被证明前扩张高级自动化。
