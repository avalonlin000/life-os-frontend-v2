## 边界

- 结衣不做电竞分析，那是小雪电竞人生。
- 结衣不做普通 todo list，任务必须服务反馈调整。
- 结衣不做普通知识库，知识必须转行动或沉淀认知资产。
- 结衣移动端优先，桌面端只是补充。
