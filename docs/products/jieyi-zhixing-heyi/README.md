## 私人系统投资总纲

> 结衣是一套长期只服务钧钧、覆盖整个人生，帮助钧钧持续认识世界、改造世界，并在实践中更新世界观和方法论的私人系统。

项目源于对当前生活的不满。要改变现实，必须先认识现实和客观规律；新的认识形成方法并进入实践，实践结果再继续修正世界观和方法论。

```text
现实状况与矛盾
  → 认识世界：事实、知识、规律、世界观
  → 改造世界：方法、时间、行动、实践
  → 实践结果与新问题
  → 再认识、再改造
  ↺
```

### 产品承诺

> 帮助钧钧更准确地认识世界，更有效地改造现实，并在实践、认识、再实践、再认识的循环中持续增强能力。

### 投资边界

- 这是钧钧的私人系统，不以通用化、多人化或商业化为目标。
- 它覆盖整个人生，但所有能力最终归到“认识世界”或“改造世界”。
- 人生方向决定哪些事情值得投入，现实课题承接当前真正需要认识和改变的事情；领域用于看全局，目标和任务只是实践工具。
- 知识库、记录和学习主要服务世界观更新；时间管理、日程、课程和实践主要服务方法论落地。
- 复盘和反馈负责用实践结果重新检验认识与方法。
- 外部项目只吸收能增强“认识—改造—再认识”循环的部分。
- 在真实主飞轮走通前，不扩张高级分析、复杂自动化或第二套入口。

### 成功标准

- 钧钧能够更准确地理解当前现实、矛盾和客观条件。
- 新认识能够形成方法并真正改变现实生活。
- 实践结果能够修订世界观和方法论，而不是只积累知识或任务。
- 现实持续改善，钧钧认识和改造世界的能力持续增强。

## 完整产品顶层设计

```text
钧钧：人生方向、价值判断、最终确认
                  ↓
现实课题：当前不满、事实、矛盾、客观条件
                  ↓
认识世界：现实记录＋人类知识 → 当前世界观与判断
                  ↓
改造世界：方法论 → 时间、行动与现实实践
                  ↓
现实结果：变化、阻力、新事实和新问题
                  ↓
反馈更新：修正世界观与方法论，继续下一轮实践
                  ↺
```

知识线重点支持认识世界，岁月时间管理线重点支持改造世界，复盘和反馈连接两者。结衣项目保存并运行长期人生系统；结衣 Agent 负责理解、调用、建议、回看和协调，是日常交互入口，不是第三套人生系统。
