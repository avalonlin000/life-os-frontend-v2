| `--text` | `#e8ddd0` | 正文 |

### 字号

| 角色 | 大小 | 用例 |
|------|------|------|
| 标题 | 14px 粗体 | 页面标题 |
| 模块标题 | 13px 大写 + 1px 间距 | section-title |
| 正文 | 14px | 知识内容、活动名 |
| 辅助文字 | 12px | 时间戳、提示、来源 |
| 标签 | 11px | 标签/分类 |

### 间距

- 页面左右边距：桌面 24px，移动端 14px。
- 区块间距：24px。
- 卡片间距：12px。
- 卡片内边距：16px。
- 标签组间距：6px。

## 2. 顶层布局

```text
page-shell
├── page-shell-sidebar
│   └── side-nav
│       ├── 知 /know
│       ├── 行 /act
│       ├── 思 /reflect
│       └── 道 /way
└── page-shell-main
    ├── header
    └── page-shell-content
```

移动端行为：

- `max-width: 720px` 时隐藏侧边栏，使用底部导航或汉堡菜单。
- 所有横向 flex 在移动端变为纵向。
- 网格布局变为单列。
- 按钮全宽，最小高度 44px。

## 3. 知页 `/know`

### 区块顺序

1. 深度学习
2. 外部知识
3. 个人记忆 / 整理洞察
4. 每日小学习
5. 导入入口

### 深度学习面板

| 区域 | 规格 |
|------|------|
| 输入区 | 主题输入框 + 范围下拉 + “生成学习包”按钮 |
| 资料结果区 | 显示真实 Wiki 材料；fallback 时只能显示“搜索接口未就绪” |
| 深挖问题区 | 3 个问题卡片，点击后高亮 |
| 学习包区 | 核心材料 / 相关旧知识 / 边界候选 |
| 五卡引导 | problem / structure / relation / boundary / action 五个 textarea |
| 验收回写 | quality level + destination + “回写验收”按钮 |

状态规则：

- fallback：黄色徽章，“接口未就绪 · 显示占位提示”；不能算真实学习完成。
- live：绿色徽章，“API 已连接”；材料必须来自真实 Wiki 检索。

## 4. 行页 `/act`

### 区块顺序

1. 今日学
2. 知识拆解
3. 今日做

### 今日学

- 展示 `dailyPlan.learn`：pillar、title、content、questions、source。
- 展示 `dailyPlan.review`：fromDate、title、snippet、question。
- 无数据时显示“今日学习内容尚未生成（凌晨4:05自动生成）”。

### 今日做

- AI 建议任务显示在“知识拆解”。
- 用户添加任务显示在“今日做”。
- 完成后：opacity 降低、line-through、按钮变为已完成。
- 再点已完成任务可重开。

## 5. 思页 `/reflect`

### 区块顺序

1. 活动记录
2. 今日复盘统一输入
3. 今日整理
4. 节奏建议
5. 历史整理 / 认知资产候选

### 今日复盘统一输入

- 前台只保留一个 textarea，不再拆“今日备注 / 明日调整 / 心情 / 思想”。
- 用户只打一段自然语言，保存到 `mood.note`。
- 保存复盘不创建 schedule，不影响 /act 今日行动队列。
- 长期模式分析先依赖约 10 天原文沉淀，不在前台增加分类选项。

### 今日整理

- 已有 `daily_review` 时展示 summary、highlights、concerns、suggestion。
- 无 review 时显示“还没有今日整理”与生成按钮。
- 生成中按钮 disabled，文案“生成中...”。
- 生成失败不能填充 mock 数据。

## 6. 道页 `/way`

`/way` 是当前「道」页主入口；`/dao` 仅保留为兼容 alias 或历史称呼，进入后跳转到 `/way`。底部导航和侧边导航都使用 `/way`，不要改成 `/dao`。

### P1 页面结构

1. 长期目标
2. 智慧卡片
3. 反复模式
4. 决策原则

道页不是 P0 闭环的阻塞项，但导航和空状态应该保留，避免用户误以为系统断层。

## 7. API 不可用 UI 规则

| 场景 | 展示 |
|------|------|
| 知识导入失败 | toast “导入失败” |
| 拆解失败 | toast “拆解失败” |
| 深度学习失败 | warning 徽章 + “搜索接口未就绪” |
| 今日整理失败 | “生成失败：今日整理接口暂未就绪” |
| 活动保存失败 | “记录失败，稍后重试” |
| Wiki 写入失败 | “写入失败，未保存为认知资产” |
