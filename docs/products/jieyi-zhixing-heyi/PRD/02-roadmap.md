# 结衣知行合一 — Roadmap

> 版本：1.0 | 2026-06-28

---

## P0：知行思闭环

1. /know 知识输入与列表稳定。
2. /know 深度学习入口可用：主题 → 搜索 → 三问 → 学习包 → 五卡回写。
3. /act 今日执行队列稳定：完成、重开、手动补充。
4. /reflect 活动记录与统一复盘输入稳定。
5. /reflect 今日整理和节奏建议显式成功/失败。
6. 认知资产写入 Wiki，有来源记录。

---

## P1：道层增强

1. /act 展示 daily_plan.review 复习卡片。
2. /way 展示长期目标；/dao 仅保留为兼容 alias/历史称呼。
3. /way 展示智慧卡片；底部导航和当前主入口仍使用 /way。
4. 后台识别反复模式。
5. Hermes cron 接管每日整理生成。

---

## 开发节奏

- 先保闭环，再做美化。
- 先保真实数据，再做 AI 生成。
- 先移动端可用，再补桌面端展示。
- API 不可用时宁可显式失败，不做假成功。
